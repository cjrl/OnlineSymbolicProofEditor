What does this editor do?
This editor provides a simple way of typesetting propositional and predicate logic proofs. I wrote this editor my freshman year of college to help quickly create great looking proofs for my discrete math and logic classes.
Use Basics:
The input box labeled "Statement to be proved" is where the overall logical statement that the proof is about goes. In the final rendering it appears above the actual proof.
The three columns under the title proof is where the actual editing of the proof occurs. The first column is for the logical statement (or LaTeX expression). The middle column is where the line numbers that were used to justify the current line belong. It is only possible to input numbers into the middle column, each number should be separated by a single space. The last box is where the logical equivalence rule that was used to derive the current line is put.
Key Controls:
Pressing ENTER while inside the editor area inserts a new line under the currently selected line.
Pressing BACKSPACE while inside the first column of the editor when the first box is empty deletes the current line.
The ARROW KEYS can be used to move around the editor columns and rows.
Pressing SHIFT ENTER renders the LaTeX expression, useful for when auto rendering is off.
The TAB key can be used to move from column to column, and on the bottom line can be used to create a new line if it is in the right column.
Creating Logic Statements:
Although the editor accepts direct LaTeX sequences the purpose of the editor was to speed up the process of creating good looking proofs. So the editor has built in symbols it accepts as short hand. They are as follows:
Input &|-><>exall~xor
Output $$\wedge$$$$\vee$$$$\Rightarrow$$$$\iff$$$$\exists$$$$\forall$$$$\lnot$$$$\oplus$$
Printing, Loading, and Saving:
To print only the proof output use the print button above the proof editor. The normal print key will print the entire page.
Loading and Saving files only works in browsers that support some of the newer HTML5 features. To open a file select the file with the browse button and then click the open button. To save a file click the save button. The file will be saved to your downloads folder, and won't override any existing files of that name due to the limitations of javascript.
